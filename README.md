***CAP MANAGEMENT SYSTEM***
This project is a Cap Management System, built using core concepts like functions, loops (for & while), if-else conditions, nested if statements, and data types.
Q) How does the Program Work?
The system starts by creating a simple user portal where users enter their name and password. This data is stored using lists and dictionaries.
Next, the program allows users to select the type of ride service (City or Outside City). Based on their choice, 
It then asks for service options such as AC Car, Non-AC Car, or Bike. This is where if and elif conditions are applied to manage multiple decision paths.
Then, the user enters the fare amount. The program processes this input and uses a for loop with conditions to match the selected service with the appropriate fare range. 
Based on this logic, it displays a customized message for the user.
The entire system runs inside a while loop, allowing multiple users to interact with the program until they choose to exit.
